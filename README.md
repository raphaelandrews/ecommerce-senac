## Run
Clone

```bash
git clone https://github.com/raphaelandrews/ecommerce-senac.git
cd ecommerce-senac
code .
pnpm install
Setup Supabase e Clerck - .env.example
pnpm dev
```

Ou vá para [Deploy](https://ecommerce-senac.vercel.app)
